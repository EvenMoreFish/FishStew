# EMFFishStew

An EvenMoreFish addon that adds items to start a fishing contest.

This addon adds a new RewardType to give a stew item. This item can start any configured competition, optionally respecting the minimum player requirement.

> [!NOTE]  
> This addon is its own plugin and will not work inside the EvenMoreFish addons folder.

## Download

Release Builds are available on:
- [GitHub](https://github.com/EvenMoreFish/FishStew/releases)

Snapshot/Dev Builds are available on [Jenkins](https://ci.codemc.io/job/EvenMoreFish/job/EMFFishStew/).

> [!IMPORTANT]  
> EMFFishStew depends on [EvenMoreFish](https://modrinth.com/plugin/evenmorefish). The plugin will not load without it.
